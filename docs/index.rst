vod-metadata
============

Introduction
------------

The `vod-metadata` project contains tools for working with CableLabs Video On
Demand (VOD) Metadata files.

Included is a command line utility that can automatically generate metadata for
video files and a Python library for manipulating metadata files.

References
----------

For more information on the VOD Metadata specification, see these documents
at the CableLabs website:

*   `VOD Metadata 1.1 - Content <http://cablelabs.com/specification/cablelabs-video-on-demand-content-specification-version-1-1/>`_
*   `VOD Metadata 1.1 - Asset Distribution Interface <http://www.cablelabs.com/specification/cablelabs-asset-distribution-interface-specification-version-1-1-2/>`_

`vod-metadata` builds on the excellent `MediaInfo <https://mediaarea.net/en/MediaInfo>`_
project - see the installation document below for instructions for installing
MediaInfo.
